### Minimum Setup
Users need **just 5 environment variables** to run production MNEMOS:
```bash
PG_HOST=localhost           # Database
PG_DATABASE=mnemos
PG_USER=mnemos
PG_PASSWORD=your_password
MNEMOS_API_KEY=$(openssl rand -hex 32)

# At least one of:
TOGETHER_API_KEY=xxx        # Free tier
# OR
GROQ_API_KEY=xxx            # Free tier
```

No GPU required. CPU-only fully functional.
